AFQ-Browser

Exploratory analysis, visualization, and sharing

Challenge

Results from large multi-dimensional datasets are hard to

  • understand
  • communicate
  • reproduce
  • share

Solution: a tool for exploratory data analysis (EDA), with data sharing built-in!

  • A web-based application
  • Leverages modern visualization frameworks
  • Builds a web-site for a diffusion MRI dataset
  • Automatically uploads the website to GitHub
